API Key de Google

Necesitas una API Key de Google Cloud para acceder a los archivos de tu Drive.

Como obtener una API Key
  1. Ve a Google Cloud Console
  2. Crea un nuevo proyecto o selecciona uno existente
  3. Ve a APIs y servicios > Biblioteca
  4. Busca "Google Drive API" y habilitala
  5. Ve a APIs y servicios > Credenciales
  6. Haz clic en "Crear credenciales" > "Clave de API"
  7. Copia la clave y pegala aqui

Carpeta de TeraBox

Pega el enlace de tu carpeta compartida de TeraBox. Las subcarpetas se convertiran en playlists.

Asegurate de que la carpeta este compartida publicamente

Como compartir una carpeta de TeraBox
  1. Ve a TeraBox e inicia sesion
  2. Crea una carpeta y sube tus archivos de musica
  3. Haz clic derecho en la carpeta > "Compartir"
  4. Selecciona "Publico"
  5. Copia el enlace y pegalo aqui
