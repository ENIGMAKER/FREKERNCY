API Key de Google

Necesitas una API Key de Google Cloud para acceder a los archivos de tu Drive. Es gratis y solo toma 5 minutos.

Como obtener una API Key (5 minutos)
  1. Ve a Google Cloud Console
  2. Crea un nuevo proyecto o selecciona uno existente
  3. Ve a APIs y servicios > Biblioteca
  4. Busca "Google Drive API" y habilitala
  5. Ve a APIs y servicios > Credenciales
  6. Haz clic en "Crear credenciales" > "Clave de API"
  7. Copia la clave y pegala aqui
  8. IMPORTANTE: En "Restricciones de aplicacion" selecciona "Ninguna" o agrega tu dominio
  9. IMPORTANTE: En "Restricciones de API" selecciona "No restringir clave" o agrega "Google Drive API"

Carpeta de Google Drive

Pega el enlace de tu carpeta compartida. Las subcarpetas se convertiran automaticamente en playlists.

Asegurate de que la carpeta este compartida como "Cualquier persona con el enlace puede ver"

Como compartir una carpeta de Drive
  1. Ve a Google Drive
  2. Crea una carpeta y sube tus archivos de musica (mp3, m4a, ogg, wav, flac)
  3. Puedes crear subcarpetas para organizar por albums/playlists
  4. Haz clic derecho en la carpeta principal > "Compartir"
  5. En "Acceso general" cambia a "Cualquier persona con el enlace"
  6. Asegurate de que diga "Lector" (no Editor)
  7. Copia el enlace y pegalo aqui
  8. NOTA: Todos los archivos DENTRO de la carpeta heredan los permisos
